## Meaning

`〜てある` describes a state that exists because someone did an action, often for a purpose.

## Usage

Use the `て` form of a transitive verb before `ある`.

| Pattern | Meaning |
| --- | --- |
| Objectが Verbて + あります | object has been done and remains so |

## Notes

This differs from `〜ている` because it often implies someone intentionally created the state.
