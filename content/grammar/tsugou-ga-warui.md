## Meaning

`都合が悪い` means that a time, date, or plan does not work for someone.

## Usage

Use a time expression as the topic, then add `都合が悪いです`.

| Pattern | Meaning |
| --- | --- |
| Time は都合が悪いです | that time is inconvenient |
| 都合が悪いので... | because it is inconvenient... |

## Notes

This is a useful soft explanation when changing appointments or declining invitations.
