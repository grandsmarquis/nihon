## Meaning

`〜はずです` means "should be..." or "is expected to..." based on information or reasoning.

## Usage

Use a plain form before `はずです`. For な-adjectives and nouns, use `な` or `の`.

| Pattern | Meaning |
| --- | --- |
| Verb plain + はずです | should do; should be |
| い-adjective + はずです | should be |
| な-adjective + なはずです | should be |
| Noun + のはずです | should be |

## Notes

This is useful when politely explaining an expectation from a reservation, schedule, or prior agreement.
