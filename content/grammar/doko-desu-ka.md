## Meaning

`どこですか` asks "where is it?" or "where is...?" It is a basic question pattern for locations.

## Usage

Put the place or thing before `は`, then ask `どこですか`.

| Pattern | Meaning |
| --- | --- |
| Noun は どこですか | Where is Noun? |
| ここ / そこ / あそこです | It is here / there / over there |

## Notes

This is useful during apartment viewings when asking about rooms, storage, entrances, and shared spaces.
