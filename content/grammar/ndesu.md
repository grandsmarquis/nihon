## Meaning

`〜んです` gives or asks for an explanation. It can make a statement sound more connected to the situation.

## Usage

Use a plain form before `んです`. For nouns and な-adjectives, use `なんです`.

| Pattern | Meaning |
| --- | --- |
| Verb plain + んです | the thing is, ... |
| い-adjective + んです | it is that ... |
| Noun / な-adjective + なんです | it is that ... |

## Notes

`〜んですが` is often used before a request because it politely introduces the reason for asking.
