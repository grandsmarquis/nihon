## Meaning

`〜まま` means that something stays in the same state.

## Usage

Use a plain form, noun + `の`, or な-adjective + `な` before `まま`.

| Pattern | Meaning |
| --- | --- |
| Verbた + まま | left as done |
| Verbない + まま | still not done |
| Noun + のまま | still noun |

## Notes

This is useful for describing unresolved problems or unchanged conditions.
